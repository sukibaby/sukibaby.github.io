ITGmania Easy Install for Linux Guide

        On Linux, it is even easier than Windows to use the game and manage your content. Nearly all the worst aspects of Windows are a non-issue with Linux:

  • No AppData
  • Portable Install By Default (when following this guide)
  • Automatically installs Arrow Cloud Theme
  • One-step method to update the game and theme

The benefits of using Linux for ITGmania actually make the game easier to interact with & upgrade, so it’s recommended even for users without Linux experience or knowledge.

Before replacing Windows with Linux, make sure you have your Songs and Save folders backed up! (Noteskins too if you have custom Noteskins)

You can use the following tool to make a Linux install USB, regardless of what OS you currently use. https://etcher.balena.io/

Another option for Windows users is Rufus: https://rufus.ie/en/

Pick a Linux variant to install. The ones I recommend are below, because they come with GPU drivers included by default.

  • Linux Mint https://www.linuxmint.com/ 
  • Includes AMD/Intel graphics drivers. NVIDIA drivers must be installed separately.
  • Pop_OS https://system76.com/pop/
  • Includes AMD, Intel and NVIDIA drivers.

Use Etcher to write the Linux installer to the USB stick, and start up on your ITG PC to install Linux. Go thru the default options in the installer to wipe the drive in your computer.  

Once you have restarted into Linux, open the Terminal.  This might be called Terminal, Konsole, LXTerminal, etc. copy paste the following commands into  the terminal and press enter after each one. The last one will take the longest, so you can leave for ~15 minutes and come back later.

wget https://sukibaby.github.io/itgmania-setup.sh

chmod +x itgmania-setup.sh

./itgmania-setup.sh

Later on, when you want to update the game or the theme to the latest version, simply open the Terminal again and type ./itgmania-setup.sh again. That is all you ever have to do!

The setup script will create a portable install, so your game content and game are ALWAYS in the same place!

These instructions were tested very thoroughly so you should not have any issues from this point, but if you have questions or run into any issues, many people on the ITC discord are around and able to help out, just be sure to give lots of details about your setup to speed up how fast you’ll get a helpful answer.

Troubleshooting

If there are graphics issues, here are some of the most common remedies.

Choppiness

Make sure Exclusive Fullscreen is used instead of Borderless Fullscreen for best performance. You can’t alt tab with Exclusive Fullscreen, though, so if you need to have that functionality, you may have to use Borderless Fullscreen.

Nvidia - Screen Tearing

Usually sync to vblank is enabled by default in OpenGL settings. If it is, disable sync to vblank.

Suggestions for non-supported Linux variants

If not using one of the suggested distributions, note that a common issue with Linux is if the keyboard works in Windowed mode, but not in Fullscreen. If this happens to you, it means you’re logged in with Wayland and need to be logged in with X11. What that means isn’t important - you can resolve this by logging out (click the Start Menu, then Log Out). On the login screen, you should see a gear or some kind of options menu in the corner.  Click that and you’ll likely see another option to login that uses X11, or an option that doesn’t mention Wayland. Click that, and log back in. That should resolve the issue for good.
